# Una-pretensión-justa
Las funcionarias pertenecientes a una de las reparticiones más importantes del Estado, la Aduana, se empeñan en estos momentos en iniciar gestiones ante los Poderes Públicos para obtener un aumento de sueldo. Se trata según lo han manifestado públicamente los iniciadores de dichos trabajos de obtener especialmente el acrecimiento de las asignaciones de menor cuantía.
